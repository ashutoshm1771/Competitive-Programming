# Sliding Window
---
## Outline
1. Technique description
2. Problem variants
    - finding the smallest window that satisfies some criteria.
      * Example: smallest window with sum >= S
    - finding the window which optimizes an objective function given a fixed length L.
      * Example: the window of length L whose sum of elements is minimum
    - finding the value of a function for each possible window of a fixed length L.
      * Example:  the minimum element in each window of length L
